How I Made the Website

This website was created with a combination of HTML, CSS, and JavaScript. The website was hosted on Github.

Steps I took to host my website on Github:

  • Login to Github
  • Create a Github repository
  • Clone the Github Repo outside of the folder containing all your files.
  • Move the folder containing all your files into the Github Repo
  • Add the files, commit, and push to Github
  • Go to Settings and scroll until you get to the Github pages section.
  • Copy and paste the link shown into a new browser tab. In my case: "krishna-tx.github.io"
  • Voila! Now you have your website running completely free on the web.

    • I used the w3schools website to get information on syntax for using HTML, CSS, and JavaScript
